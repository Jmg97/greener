# salady

<salady> 홈페이지 제작
* 보러가기 : (http://salady.com/)

# 프로젝트 목표

javaScript와 media query를 이용한 <salady> 반응형 홈페이지 제작

# 프로젝트에서 사용한 기술

JavaScript ES6

라이브러리를 사용하지않고 바닐라 Js로 작성

# 반응형 웹페이지

* Pc,타블렛,모바일 등 해상도에 따라 보이도록 미디어 쿼리 작성
